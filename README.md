# chess-tracker-backend
made with Express and Typescript

### Routes
POST /login

GET /boards?matchId={matchId}\
GET /boards/{boardId}\
PUT /boards/{boardId}

GET /matches\
GET /matches?all=true\
GET /matches/{matchId}\
POST /matches\
DELETE /matches/{matchId}

GET /schools\
POST /schools\
DELETE /schools/{schoolId}

GET /users\
POST /users\
DELETE /users/{userId}
